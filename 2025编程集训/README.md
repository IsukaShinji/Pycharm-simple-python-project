# 2025 暑假编程集训

本目录整理了学院 2025 年暑假编程集训期间完成的代码。集训目标是从网页抓取、URL 规范化、文本预处理、倒排索引到 Web 检索界面，逐步实现一个小型站内搜索引擎原型。

项目最终形态是一个面向人大新闻或校内网页数据的检索系统：先抓取网页并保存 HTML，再对页面内容进行清洗和索引，最后通过 Flask 页面输入关键词并返回相关结果。

## 项目内容

- URL 处理：练习相对路径还原、链接规范化、重复链接过滤。
- 网页爬取：限制目标域名，使用请求头、超时和爬取间隔控制访问过程。
- 数据保存：将抓取到的 HTML 文件保存到本地，作为后续检索语料。
- 文本预处理：提取标题、正文、链接等字段，清理页面噪声。
- 检索实现：构建基础倒排索引和关键词匹配逻辑。
- Web 展示：使用 Flask 提供搜索框和结果页。

## 技术栈

- Python
- requests
- BeautifulSoup
- url-normalize
- Flask
- 基础信息检索方法：分词、倒排索引、关键词匹配、相关性排序

安装常用依赖：

```bash
pip install requests beautifulsoup4 url-normalize flask
```

## 运行方式

如果只想查看 Web 检索原型，可进入 `草稿/` 目录后运行：

```bash
python app.py
```

然后访问本地 Flask 服务。不同脚本使用的数据目录不完全一致，复现时需要确认 HTML 数据路径和索引文件路径。

## 复盘

这个项目让我第一次把“爬虫 + 文本处理 + 检索 + 前端展示”连成一个完整流程。当前版本能完成基础检索，但相关性排序仍有提升空间，尤其是关键词权重、标题加权、BM25 参数调节和中文分词质量。它更像一个信息检索系统的最小可用原型，而不是成熟搜索引擎。
