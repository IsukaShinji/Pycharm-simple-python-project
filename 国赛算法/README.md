# 数学建模与机器学习算法练习

本目录收录了我在备战数学建模竞赛期间整理和复现的一组算法练习。每个脚本都围绕一个典型算法或建模方法展开，通常采用“模拟数据或示例数据 -> 建模 -> 计算 -> 输出结果/可视化”的结构，便于快速理解算法能解决什么问题、输入输出是什么、代码如何落地。

这些代码适合作为数学建模竞赛前的算法工具箱：不追求工程封装，而是强调模型思想清楚、注释充分、能快速改造成比赛中的子模型。

## 覆盖算法

### 机器学习分类与预测

- K 近邻：鸢尾花分类。
- 朴素贝叶斯：垃圾文件判断。
- 决策树：员工离职预测。
- 随机森林：日用品周销量预测。
- 支持向量机：红酒分类。
- 二分类模型：信用卡欺诈识别。

### 聚类分析

- K-Means：顾客分类。
- 肘部法选择 K：鸢尾花聚类。
- 层次聚类：肿瘤样本分类。

### 优化算法

- 模拟退火：二维多峰函数最小化。
- 粒子群优化：线性参数寻优。
- 自适应方法：信用卡相关示例。

### 评价、预测与随机过程

- 模糊综合评价：供应商选择。
- 灰色预测：企业能耗预测。
- 马尔可夫链：股票状态预测。
- 排队论：柜台服务系统分析。
- 强化学习：网格迷宫路径策略。

## 运行说明

常用依赖：

```bash
pip install numpy pandas matplotlib scikit-learn scipy
```

部分脚本使用随机模拟数据，每次运行结果可能略有不同。阅读时建议先关注脚本开头的数据构造和参数设置，再看模型训练、评价指标和图像输出。

## 复盘

这个目录的定位是“竞赛算法速查本”。数学建模中经常需要在有限时间内判断某个问题适合预测、分类、评价还是优化，因此把常见算法整理成短脚本，有助于在正式比赛中快速迁移和组合。
