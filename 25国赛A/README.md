# 2025 全国大学生数学建模竞赛 A 题

本目录收录了 2025 年国赛 A 题的建模与代码实现。项目主题是无人机投放烟幕干扰弹后的遮蔽效果计算与策略优化，需要在三维空间中描述导弹、无人机、烟幕云团和真实目标之间的几何关系，并在约束条件下寻找更优的投放方案。

我在队伍中主要负责程序实现与数值验证。代码中保留了多版求解脚本，既包括直接几何仿真，也包括面向多变量策略搜索的优化版本，体现了从“验证单个方案”到“搜索最优方案”的建模推进过程。

## 项目内容

- 问题一：根据给定无人机速度、投放时间和起爆延迟，计算烟幕对导弹 M1 的有效遮蔽时长。
- 问题二、三：扩展投放参数，搜索更优的飞行方向、速度、投放时间和起爆时间。
- 问题四、五：处理多无人机、多导弹或多烟幕弹协同下的复杂优化问题，并进行方案验证。

## 核心方法

1. 三维运动建模：用向量形式描述导弹匀速飞行、无人机等高度飞行、干扰弹平抛运动和烟幕云团下沉过程。
2. 目标离散化：将真实目标圆柱体按高度层和圆周方向采样，避免只判断中心点造成遮蔽时间虚高。
3. 遮蔽判定：将“导弹到目标采样点”的视线段与烟幕球体做相交判断，所有关键采样点被遮挡时才计入有效遮蔽。
4. 数值积分：按时间步长扫描烟幕有效期，累计满足遮蔽条件的时间。
5. 参数优化：在不同脚本中尝试启发式搜索、分段验证和多模型交叉检查，提升方案稳定性。

## 运行说明

主要依赖：

```bash
pip install numpy scipy matplotlib
```

多数脚本可直接运行，但不同版本中的参数、时间步长和采样精度并不完全一致。阅读时建议先从 `25A第一问（1）.py` 理解遮蔽判定逻辑，再看后续优化脚本。

## 复盘

这个项目最大的收获是几何判定必须严格服务于题意。早期如果只判断目标中心点，很容易得到看似漂亮但不可靠的遮蔽时长；改为圆柱面多点采样后，结果更保守，也更接近题目要求。竞赛后期也暴露了代码版本管理的重要性：多成员协作时，关键代码和实验结果必须及时归档。
