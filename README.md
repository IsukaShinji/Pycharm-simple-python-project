# Simple Python Project

这里整理的是我在课程学习、数学建模竞赛、暑期集训和自学算法过程中写下的 Python 代码。仓库不是一个单一的软件工程，更像是一个持续更新的学习与项目档案：有竞赛建模脚本，也有课程实验、算法复现、爬虫检索系统原型和一些阶段性片段。

我希望这个仓库能在个人网站上承担两个作用：一是记录自己做过什么，二是让外部读者能快速看懂每个目录的大致主题、技术路线和完成度。后续新的课程项目、竞赛代码和更完整的材料会继续补进来。

## 仓库特点

- 以 Python 为主，覆盖 `pandas`、`numpy`、`scikit-learn`、`scipy`、`matplotlib`、`PyTorch`、`Flask` 等工具。
- 代码大多来自真实课程或竞赛场景，保留了从初版、补丁、验证脚本到最终方案的演进痕迹。
- 很多脚本偏研究和建模用途，不是生产级封装；部分早期代码仍依赖本机路径或比赛附件数据。
- 每个重点目录会逐步补充目录级 `README.md`，说明问题背景、技术路线、文件结构和复盘。

## 如何阅读

如果你想看数学建模竞赛项目，可以从 `23国赛C/` 和 `25国赛A/` 开始；如果想看算法复现，可以看 `国赛算法/`；如果想看课程实验，可以看 `25秋人工智能引论/`；如果想看一个从爬虫到检索页面的完整小项目，可以看 `2025编程集训/`。

由于部分项目使用竞赛附件、课程数据或本地生成文件，直接运行前需要先检查脚本中的数据路径和依赖环境。目录级 README 中会尽量标出这些限制。
