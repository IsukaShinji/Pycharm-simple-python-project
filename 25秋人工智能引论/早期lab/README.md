# 早期实验

本目录保存人工智能引论课程前期的基础算法实验，主要用于练习经典算法思想和简单数值模拟。

## 文件导览

| 文件 | 说明 |
| --- | --- |
| `lab2_prunning.py` | 剪枝相关实验。 |
| `lab3-蒙特卡洛法求圆周率.py` | 使用蒙特卡洛方法估计圆周率。 |
| `lab4-K-means.py` | K-Means 聚类实验。 |

## 使用方式

这些脚本通常可以单独运行：

```bash
python lab3-蒙特卡洛法求圆周率.py
```

不同脚本的依赖不完全一致，运行前可先查看文件顶部的 import 列表。
