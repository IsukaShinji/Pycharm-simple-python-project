# 英中翻译数据

本目录保存 Transformer NMT 实验使用的英中句对数据。

## 文件导览

| 文件 | 说明 |
| --- | --- |
| `cmn.txt` | 原始英中平行语料。 |
| `train.txt`、`dev.txt`、`test.txt` | 训练、验证和测试数据。 |
| `train_mini.txt`、`dev_mini.txt`、`test_mini.txt` | 小规模调试数据，用于快速验证代码流程。 |

## 使用提示

调试模型结构或数据预处理时优先使用 mini 数据，正式训练再切换到完整数据。文件格式和读取逻辑由上级目录中的训练脚本约定。
