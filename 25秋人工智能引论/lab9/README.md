# Lab 9 音频实验

本目录保存人工智能引论课程的音频实验材料，包括 notebook、实验 PDF 和示例音频文件。

## 文件导览

| 文件 | 说明 |
| --- | --- |
| `lab9.ipynb` | 音频处理实验 notebook。 |
| `lab9.pdf` | 实验说明或导出文档。 |
| `piano_c.wav` | 钢琴 C 音示例音频。 |
| `scale.wav` | 音阶示例音频。 |
| `readme.txt` | 原始简要说明。 |

## 使用提示

建议使用 Jupyter 打开 `lab9.ipynb`。运行前确认 notebook 中引用的音频文件路径仍指向当前目录。
